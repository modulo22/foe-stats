Usage

This is the usage for Google Chrome browser.

Open developer tools

Go to forgeofempires.com and connect you to your world.

  • 1. Open the menu
  • 2. Click on More tools
  • 3. Click on Developer tools

Filter requests

  • 1. Click on Network
  • 2. Enter the filter json
  • 3. Reload the page with the button in the left near the address bar

Copy requests

  • 1. Click on a request filtered
  • 2. Click on Response
  • 3. Select the response with click and Control+A
  • 4. Copy the response with Control+C

Update statistics

In this page

  • 1. Click on the text field and copy the response with Control+V
  • 2. Click on Update
  • 3. Repeat the operation with all the json for having all check OK

In order to get all JSON, you may need to open Event history and display friend list, guild list and neighbors.

You can see Tavern sit and Polish/motivate in the other tabs.
